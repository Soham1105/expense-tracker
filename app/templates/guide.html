Feature Guide Everything this app can do

Core Workflow

upload_file

Import Bank Statement

Upload CSV bank statements via the upload page. Transactions are auto-deduplicated, vendor names normalised, and tag rules applied automatically on import.

Hidden: Duplicate rows across re-uploads are silently skipped.

sell

Tag Transactions

In Reports, select transactions and bulk-tag, or click a transaction to tag it individually. Tags propagate to counterparty-matched transactions automatically.

Hidden: Manually tagged = "manual" badge. Auto-tagged by system = "auto" badge.

rule

Tag Rules → open

Create rules like "Vendor contains Zomato → tag Outside-food". Rules run on every upload automatically. Hit "Apply Rules Now" to run against all existing transactions.

Supports: Contains, Exact, Regex. Fields: Vendor, UPI ID, Narration, Counterparty Name, Payment Mode.

call_split

Split Expenses

Mark any transaction as a shared expense. Assign amounts to people. Link recovery transactions (friend paid back). Edit recovery amounts inline via ✎ button.

Hidden: Split recovery transactions inherit all tags from the parent expense automatically.

Analytics → open

Watchlist

Pin any tag/category to your watchlist. Cards show net spend (expense + invested − income recovered). Set a monthly budget per card — green/amber/red progress bar. Clear budgets with the "Clear" button in the budget modal.

Hidden: "+ budget" on each card. Income from recovery transactions auto-subtracts.

Watchlist Modal (click card)

Deep-dive per category: monthly trend chart, top merchants bar chart, subcategory doughnut chart (2-level), P&L showing income vs expense vs invested.

Hidden: 6 time presets (1m/3m/6m/1y/2y/all). "View All" jumps to filtered Reports.

Month-over-Month Comparison

Click "Load Comparison" at the bottom of Analytics. Table shows This Month | Last Month | 3-Month Avg | Change% with ▲▼ arrows per category.

Recurring Transactions

Auto-detected patterns from the last 6 months. Highlights merchants with ⚠ "Not seen this month" in amber — spot missed expected transactions. Toggle "Debits only" to filter. Click any row to open counterparty profile.

Groups & Balances

group_work

Transaction Groups → open

Group related transactions together (e.g. gold purchases + friend contributions). Types: SPLIT, RETURN, GENERAL, CIRCLE. Balance = DEBIT roles − CREDIT roles − auto-recovered splits − settlements.

Hidden: CIRCLE type for chit-fund/investment circles. Roles: CONTRIBUTION_IN/OUT, PAYOUT_IN. Link from Reports page via the violet group badge button.

balance

Friend Balances → open

Per-person summary of who owes you what, based on split line items. Click any person card → right panel shows every transaction with outstanding amounts. Filter: "They owe me" / "I owe them".

Hidden: "Keep" person shows as fully settled (₹560 owed, ₹560 recovered).

Organisation

Category Manager

Create hierarchical categories with up to 2 levels of subcategories. Categories appear in the watchlist modal's subcategory doughnut chart.

Hidden: Categories auto-create corresponding system tags so they appear in Reports.

Categories & Tags → open

Create reusable manual tags. Merge two tags: all transactions from the source tag are re-pointed to the target, source is deactivated. Useful to consolidate "lunch" + "Lunch" + "lunch - dinner".

Hidden: Merge is irreversible — shows a confirm dialog first.

Planning

Monthly Budget → open

Set overall monthly income & expense targets. Per-category budgets shown on watchlist cards with colour-coded progress bars (green → amber → red).

Hidden: "Clear" button in budget modal removes a category budget. Budget resets each month (set per month).

Goals & Wishlist → open

Track savings goals and wishlist items with target amounts and dates. Priority levels: wishlist, nice_to_have, important, critical.

Hidden & Power Features

Things that work but aren't obvious:

  • • Counterparty profile panel — In Analytics, click any merchant in the Recurring section to see all transactions with that counterparty in a slide-out panel.
  • • Tag rules auto-run — Tag rules run automatically on every CSV upload. No need to manually click "Apply Rules".
  • • Recovery auto-tag — When you link a recovery transaction (friend pays back), it inherits all tags from the parent expense automatically.
  • • Group auto-recover — When a group's expense transaction has split recoveries, they're automatically deducted from the group balance without needing to add them as CONTRIBUTION_IN.
  • • Investment tracking in watchlist — Any tag with investment_buy transactions shows an "inv" badge. Net spend = expense + invested − income (friend contributions).
  • • Reports → Group linking — In any transaction row, click the violet group button to add it to a group without leaving Reports.
  • • Reports last-month filter — "Last Month" filter in Reports auto-sets from/to dates and activates when navigating from Analytics "Last Month" KPI.
  • • Edit split recovery amount — In the split panel, existing recovery chips have a ✎ edit button to fix the recorded amount without deleting and re-linking.
  • • Tag merge safety — Merging tags skips duplicate assignments (if a transaction already has the target tag, the source tag is just removed — no double-tag created).
  • • Subcategory modal chart — Opening a category in the watchlist shows a two-ring doughnut: outer = subcategories, inner = sub-subcategories. Click slices to navigate.
